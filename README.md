# Wingspan Score Calculator

[LIVE HERE!](https://htmlpreview.github.io/?https://github.com/luteron6/wingspan-score-calculator/blob/main/wingspan_score_calculator.html)

A lightweight, browser-based score calculator for the board game Wingspan.

The app is a single HTML file with no build tools and no dependencies. Open it in any modern browser or host it on a webserver.

## Features

- Tracks all standard end-game score categories:
- Automatically updates totals as you type.
- Supports up to 5-player scoring on desktop.
- Mobile-friendly mode switches to scoring a single player. Rotate device horizontally for 5-player support.

## Installation

A couple options:
1. NO INSTALLATION! Visit the live webpage [here](https://htmlpreview.github.io/?https://github.com/luteron6/wingspan-score-calculator/blob/main/wingspan_score_calculator.html). That's it!

2. Pull this repo and run the .html file.

3. Pull this repo and run it locally by opening it in your web brower.
